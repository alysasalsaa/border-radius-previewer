# border-radius-previewer

A real-time CSS border-radius previewer built with HTML, CSS, and JavaScript.

## Features

- [x] Preview a box with border-radius applied
- [x] Change 4 border-radius values independently (top-left, top-right, bottom-left, bottom-right)
- [x] Copy the resulting CSS to clipboard
- [x] **Bonus:** Toggle 8-value mode for complex shapes

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## How to run

Open `index.html` in your browser — no build tools needed.
